# 👋 Link Kısaltma Apisi (web)

Golang ve MySql ile yazılmış bir link kısaltma sistemidir.

### 📥 Kurulum

Kurulum için terminale şu kodları yazın : 

```bash
go get 
```

### 🧨 Başlatmak

Komut satırına bunu yazın :

```bash
go run .
```
